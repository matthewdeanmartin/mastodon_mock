Hosting

This client is a static site hosted on GitHub Pages. There is no application backend of our own; the app talks directly to whichever Mastodon server you point it at.

Inspiration

emoji-mart is the same picker Elk uses — thanks to the Elk team for showing how it should look. Keyboard shortcuts follow mastodon.social (press ? for the list), so there's nothing new to memorize.

Privacy

This site uses GoatCounter, a lightweight, open-source hit counter, to get a rough idea of how many people visit and which pages they land on. GoatCounter is designed to be privacy-friendly: it sets no cookies, does not track you across sites, and does not collect or store personal data. It records only aggregate, anonymous page-view counts — no advertising, no profiles, nothing sold to anyone. See goatcounter.com for exactly what it does and how it is implemented.

Everything else — your login, preferences, drafts, and local settings — stays in your own browser's storage. It is not sent to us; we don't run a server that could receive it. Requests for posts, timelines, and the like go straight to whichever Mastodon server you are using, under that server's own privacy policy.
