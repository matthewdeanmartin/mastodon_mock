Credits

This client is built on open-source software that ships with every page:

  • Angular — the application framework (core, router, forms; MIT license).
  • RxJS — reactive streams powering all API calls (Apache-2.0 license).
  • emoji-mart — the emoji picker, with its @emoji-mart/data emoji set (MIT license).
  • tslib — TypeScript runtime helpers (0BSD license).

emoji-mart is the same picker Elk uses — thanks to the Elk team for showing how it should look. Keyboard shortcuts follow mastodon.social (press ? for the list), so there's nothing new to memorize.
